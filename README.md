# N225 Equity Statarb — Classical vs. k-Means Clustering + LLM News-Screening Layer

This repository documents an empirical statistical-arbitrage study on the Nikkei 225, comparing 2 pair-selection methods under identical signals, position sizing, and transaction costs: classical industry classification and unsupervised k-Means clustering on return correlations. Across 589 trading days (January 2023 – May 2025, 188 constituents), industry classification produced a portfolio total return of +1.24% (avg pair Sharpe 0.083, 3/5 pairs winning), while k-Means produced −0.42% (avg pair Sharpe 0.002, 1/5 pairs winning) — evidence that shared fundamental drivers, not statistical co-movement alone, underpin durable spread relationships.

Note on the Sharpe figures: the reported Sharpe ratios are **average pair-level Sharpes** — the mean of the individual Sharpe ratios across the 5 selected pairs — not the Sharpe of the combined portfolio equity curve, which is a distinct (and generally different) quantity.

The study also surfaces a failure mode common to both methods. The highest-scoring pair, Sojitz (2768.T) / Sumitomo (8053.T), structurally decoupled after a Sumitomo copper-mine acquisition — a break visible in news text but not in price data, so no pair-selection method could have caught it. The paper's 2nd contribution proposes an LLM news-screening layer that intercepts the pipeline between z-score breach and position entry, flagging structurally-driven divergences before they destroy capital, with a narrow, testable claim and an honest baseline for when to remove it. Full detail is in `N225_Statarb_Belford_May25.pdf`.
